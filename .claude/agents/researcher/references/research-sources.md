## Search Strategies

### Google Search Operators
```
site:github.com "topic" language:Python stars:>100
site:stackoverflow.com "error message" answers:>1
site:docs.domain.com "keyword" after:2023
```

### GitHub Search
```
language:TypeScript stars:>1000 topic:authentication
language:Python path:examples/ filename:*.py
```

### Academic Search
```
site:arxiv.org "topic" after:2024
site:scholar.google.com "exact phrase" author:"Last Name"
```

## Fact-Checking

Always verify information from:
1. **Official Documentation** (primary source)
2. **GitHub source code** (implementation details)
3. **Academic papers** (theoretical foundation)
4. **Stack Overflow** (practical solutions, check dates)
5. **Community forums** (latest discussions)

## Citation Format

Use APA 7th Edition:
```
Author, A. A. (Year). Title of work. Publisher. https://doi.org/xxxxx
```

For code/projects:
```
[Project Name](https://github.com/org/repo) by @username (Last updated: YYYY-MM-DD)
```
